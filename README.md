# 🌉 LocalBoostSF

LocalBoostSF is a web application designed to help people discover small and local businesses in San Francisco. The goal is to support local communities by making it easier for users to browse, search, and interact with nearby businesses.

---

## 🚀 Features

- 🔍 Search local businesses
- 🗂️ Filter by categories (cafes, burgers, services, italian)
- 🧭 Simple and clean browsing experience
- 🏪 Business cards showing name and location
- 🔐 Login / Sign up modal (UI implemented)
- 📊 Dashboard (UI ready for future data integration)
- ⚡ Fast frontend built with React + Vite
- 🐍 Backend API built with Flask + SQLite

---

## 🛠️ Tech Stack

**Frontend**
- React
- Vite
- CSS

**Backend**
- Python
- Flask
- SQLite
